# Development Environment

- Any iOS or Android phone for testing on mobile
- Tablet and/or PC for upscaled UI testing

- PhpStorm for backend php development
- SQL shell terminal for PostresQL

- Javascript
- React

- Vite
- PHP
- Laravel
- Python
- PyTorch
- Tensorflow JS
- PostgresQL
